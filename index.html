Shortcuts

  • Ctrl/Cmd+K — Focus search
  • Enter — Search / open URL
  • Shift+Enter — Search expression (if calc)
  • Esc — Clear or unfocus
  • ? — Toggle this panel

Engines & Prefixes

  • Engines: e, g, ddg, brave, bing, w, yt, git, x
  • Force engine: "g query"
  • Site search: "@mdn array methods", "@so javascript map", "@npm vite"
  • Calculator: "= 2^10 + 5%", or "calc (12*3)/5" (Enter copies result)
